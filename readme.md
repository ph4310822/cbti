CBTI(Crypto MBTI) 产品文档

维度一：
E/S/B/T人，分别代表EVM/Solana/Bitcoin/Tron这4个有代表性的代币

维度二：风险偏好与操作频率 —— D (Degen) vs H (Hodler)
这一维度决定了你的“体质”是短线冲狗，还是长线布局。

D (Degen - 赌狗型)： * 信仰“搏一搏，单车变摩托”。

热衷于冲土狗、玩杠杆、参与高风险的流动性挖矿。

操作频繁，每天不看盘就难受，是 Gas Fee 的主要贡献者。

H (Hodler - 囤币型)： * 信仰“时间是金钱的朋友”。

以现货为主，不看短期的波动，信奉 Diamond Hands。

操作频率极低，甚至买完就删 APP，主打一个稳健。

维度三：决策依据与信息来源 —— R (Researcher) vs F (Fomo)
这一维度决定了你买入一个币的理由。

R (Researcher - 研究员型)： * 看重基本面。买币前必读 Whitepaper，研究 Tokenomics、TVL、技术架构和 VC 背景。

相信逻辑和数据，是专业的”研报复读机”。

F (Fomo - 情绪型)： * 看重社区共识和”模因（Meme）”。

信息来源主要是推特 KOL、电报群里的”Moon”表情包，或者是马斯克的一条推文。

相信”信则有，不信则无”，认为社区的情绪和热度才是最大的 Alpha。

维度四：交互习惯与信仰底色 —— N (Native) vs C (CEXer)
这一维度决定了你如何与 Web3 世界交互。

N (Native - 链上原住民)： * 非对称加密的拥趸，信奉“Not your keys, not your coins”。

钱包里常备各种链的 Gas，熟练使用 DEX、跨链桥、和各种协议打交道。

享受掌握主权的感觉，即便被科学家夹了也认。

C (CEXer - 交易所用户)： * 追求效率与便捷。认为 Binance/OKX 的 UI 才是人类文明之光。

不习惯复杂的私钥管理，所有的资产都在交易所里。

买币主要看有没有上大所，对链上繁琐的操作感到头大。

结合你的第一维度，我们来看看几种典型的”币圈人格”：
S-D-F-N (Solana - Degen - Fomo - Native)： * 典型画像： Pump.fun 的重度成瘾者。

每天在 Solana 上冲最土的狗，只看推特热度，熟练使用各种 TG 交易机器人。

B-H-R-C (Bitcoin - Hodler - Researcher - CEXer)： * 典型画像： 传统的比特币信徒。

只买大饼，研究宏观经济和减半周期，资产全存在大交易所或者硬件钱包里，几年不动一次。

E-H-R-N (EVM - Hodler - Researcher - Native)： * 典型画像： DeFi 老玩家 / 以太坊精英。

研究各种 L2 扩容方案，在 Uniswap 提供流动性，在 Aave 质押，满脑子都是技术协议和可组合性。

T-D-F-C (Tron - Degen - Fomo - CEXer)： * 典型画像： 资金盘/支付老手。

主要用孙哥的链转账或玩些简单的波场项目，追求快进快出，简单直接。


这套 CBTI（Crypto MBTI）测试题的设计思路是：通过 8 道场景模拟题，每 2 道题决定一个维度。第一维度的公链信仰（E/S/B/T）采用权重计分制，其余三个维度采用 A/B 二选一计分制。第一部分：CBTI 灵魂测试题1. 公链信仰维度 (E/S/B/T)Q1：如果你被困在荒岛，只能保留一个链上钱包地址，你会选择：A. 一个存满蓝筹 NFT 和各种协议代币的 Metamask 地址 (E +2)B. 一个交易飞快、总是能冲到最新 Meme 的 Phantom 地址 (S +2)C. 一个存放在硬件钱包里、几年没动过的“冷”地址 (B +2)D. 一个主要是为了低成本转 U 或玩老牌 DApp 的 Tron 地址 (T +2)Q2：你认为区块链最核心的魅力应该是？A. 无限的可组合性与正统的去中心化生态 (E +2)B. 极致的性能、极低的转账费与高频的交互体验 (S +2)C. 绝对的价值存储属性与宏观经济的抗通胀性 (B +2)D. 极高的转账效率与最广泛的商业落地/出入金便利 (T +2)2. 风险偏好维度 (D vs H)Q3：你刚买入的一个币在 2 小时内跌了 30%，你的反应是？A. 正常，这只是波动，甚至考虑在 CEX 挂单补仓 (H)B. 只要没归零就接着舞，或者直接割肉去冲下一个翻倍机会 (D)Q4：你的持币周期通常是？A. 以年为单位，信仰穿越牛熊 (H)B. 以小时或天为单位，翻倍就出本，甚至半小时就完成换手 (D)3. 决策依据维度 (R vs F)Q5：你在决定买入一个新项目前，首先会看？A. 官网、白皮书、经济模型、融资背景和 GitHub 提交记录 (R)B. 推特讨论热度、KOL 是否喊单、有没有好玩的 Meme 梗 (F)Q6：当市场暴跌时，你更倾向于相信？A. 链上数据（如大户流向、交易所净流入）与技术指标 (R)B. 社区的情绪、恐慌指数以及推特上的”家人们”是否还在 (F)4. 交互习惯维度 (N vs C)Q7：你大部分的资产目前存放在？A. 个人助记词钱包（如 Metamask, Ledger, Phantom）(N)B. 顶级中心化交易所（如 Binance, OKX, Bybit）(C)Q8：你如何看待“私钥/助记词”？A. 它是我的生命线，我享受完全掌控资产的安全感，即便操作很繁琐 (N)B. 太麻烦了，我更害怕弄丢它，我更相信大交易所的安保和便捷的 UI (C)第二部分：计算逻辑（算法）你可以根据测试者的选项，按以下步骤得出结论：Step 1: 确定第一主维度 (E/S/B/T)统计 Q1 和 Q2 的得分：E 总分最高 $\rightarrow$ E (EVM 派)S 总分最高 $\rightarrow$ S (Solana 派)B 总分最高 $\rightarrow$ B (Bitcoin 派)T 总分最高 $\rightarrow$ T (Tron 派)(若出现平分，Q1 的选择权重更高)Step 2: 确定后三个维度Q3+Q4： 选 A 多为 H (Hodler)，选 B 多为 D (Degen)。Q5+Q6： 选 A 多为 R (Researcher)，选 B 多为 F (Fomo)。Q7+Q8： 选 A 多为 N (Native)，选 B 多为 C (CEXer)。第三部分：典型人格举例最后你可以给用户一个类似这样的结果页面：你的 CBTI 类型是：S-D-F-N (Solana 闪电赌徒)性格特征： 你是链上的“快枪手”，对速度有近乎偏执的要求。你不需要复杂的白皮书，只需要一个极具传播力的 Meme 梗。你的金句： "Sleep is for the weak, Sol is for the moon."建议： 注意休息，你的 F5 键快被按坏了。


1. EVM 家族 (E---)：以太坊精英与 DeFi 农民这一族群通常关注底层协议、L2 扩展和复杂的链上生态。编码称号特征描述E-H-R-NDeFi 架构师硬核原住民，只在链上操作，研究深度极高，长期持有蓝筹协议。E-H-R-C以太坊合规官相信以太坊基本面，由于安全考虑资产存放在 CEX，看重研报。E-H-F-NNFT 收藏家早期蓝筹 NFT 的 Diamond Hands，靠社区共识和审美持币。E-H-F-CL2 信仰者看好以太坊生态未来，但在交易所买入后就不再动弹的“养老”用户。E-D-R-N复利收割机职业农民（Yield Farmer），在各种复杂协议间套利，研究代码漏洞。E-D-R-C打新专业户专门在交易所寻找优质 EVM 项目进行短线波段，看重机构背书。E-D-F-N链上领航员热衷于在链上寻找新的 Alpha，跟着社区情绪快速切入热门 L2。E-D-F-C小币种投机客在 CEX 里高杠杆交易以太坊系小币种，纯看盘面和情绪。

2. Solana 家族 (S---)：极速信徒与 Meme 战神这一族群追求速度、高频和极致的交互体验。编码称号特征描述S-H-R-N生态奠基人长期看好 Solana 性能，深度参与节点质押，研究并行计算等技术。S-H-R-C高性能信徒认同 SOL 的技术价值，但在 CEX 进行定投，不习惯管理私钥。S-H-F-N社区精神领袖长期持有 Solana 生态的核心 Meme，是链上活跃的社区大使。S-H-F-CSOL 持币大户信仰“SOL 必超 ETH”，单纯在交易所里囤货的稳健派。S-D-R-N科学家/夹子编写脚本在链上抢开盘，通过数据分析进行高频交易的专业选手。S-D-R-C波段交易员针对 Solana 生态币种在交易所做日内交易，极其看重盘口数据。S-D-F-NPump.fun 赌神在链上冲最土的狗，只信情绪，快进快出，Phantom 钱包重度使用者。S-D-F-C闪电投机者哪里有热度就去交易所冲哪里的 SOL 系代币，全凭直觉，不看逻辑。

3. Bitcoin 家族 (B---)：数字黄金与宏观主义者这一族群通常是行业的基石，信奉绝对的去中心化。编码称号特征描述B-H-R-N中本聪传人极致的比特币原教旨主义者，硬件钱包冷存储，只看宏观模型。B-H-R-C机构追随者通过交易所或 ETF 长期持有大饼，定期阅读行业深度研究报告。B-H-F-N铭文守望者长期持有 Ordinals 或早期铭文，相信比特币生态的文化共识。B-H-F-C大饼养老族不懂技术也不看行情，只知道“大饼是好东西”，买完就存交易所。B-D-R-N生态掘金者在比特币 L2 或协议（如 Runes）里寻找技术套利机会的冒险家。B-D-R-C杠杆套利员在交易所利用大饼的波动做高杠杆对冲或套利，极其理智。B-D-F-N铭文投机客疯狂在链上打各种新协议的铭文，全看社区喊单力度。B-D-F-C减半博弈者在交易所赌大饼波动，喜欢在重要节点通过期权、合约博取暴利。

4. Tron 家族 (T---)：实用主义与支付极客这一族群看重效率、现金流和现实世界的渗透力。编码称号特征描述T-H-R-N链上金库守卫熟练使用 Tron 钱包，研究波场质押收益，长期持有并管理资产。T-H-R-CTRX 价值投资者在 CEX 长期持有 TRX 或关联代币，看重波场生态的盈利能力。T-H-F-N波场老玩家活跃在波场早期的 DApp 中，对孙哥的生态有种莫名的“情怀”。T-H-F-C稳定币搬运工把波场当成 U 存储器，在交易所长期存放 USDT，追求极致稳定。T-D-R-N智能地址猎人在链上追踪波场大户（Smart Money）动向，跟随其进行操作。T-D-R-C套利交易商在交易所利用波场系代币的价差或资金费率进行快速套利。T-D-F-N土狗挖掘机(T版)专门在波场链上找新的 Meme 或小项目冲，极其敏锐。T-D-F-CTRC20 投机客追求转账快、手续费低，在交易所频繁买卖波场系热门代币。
